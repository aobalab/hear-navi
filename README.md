# HearNavi

A React-based web application built with Vite.

## Development

Install dependencies:
```bash
npm install
```

Run the development server:
```bash
npm run dev
```

## Build

Build for production:
```bash
npm run build
```

Preview production build:
```bash
npm run preview
```

## Lint

Run ESLint:
```bash
npm run lint
```
